<article>

Se usa para encapsular contenido independiente como publicaciones de blog.

<article>Contenido del blog</article>

<section>

Define una sección temática del documento, como capítulos o encabezados.

<section><h2>Noticias</h2></section>

<nav>

<nav><ul><li><a href="#">Inicio</a></li></ul></nav>

<header><h1>Título</h1></header>

Representa el pie de una sección o página, como información de derechos o enlaces.
